Navigation

A navigation allows users to quickly visit any section within the site or multiple sections within a container.

Subcomponents

  • Buttons
  • Pagination
  • Tabs

Pagination

View Demo
Design

Usage:

Pagination is composed of item count, back and next arrows, and page buttons. Item count is formatted to display the range of items on the current page and the total number of pages.

If there are only 1 page of results, the pagination arrows are hidden but item count is displayed. The more icon (...) is used when there are over 5 pages

Pagination should appear beneath the table or at the bottom of the page of content that it is related to.

Tabs

View Demo
Design

Usage:

Tabs allows you to navigate between multiple documents in a single window. Instead of a single title bar, a tabbed window may include multiple tabs along the top. Clicking one of the tabs will display the contents of the corresponding document.
