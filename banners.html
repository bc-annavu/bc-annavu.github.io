Banners

Banners are similar to Alerts, in that they are intended to notify the user of a contextually relevant piece of information.

While this might not be a warning or error message, it could also be used for advertising or usability-related information which may help the user improve their experience within the app.

CSS Classes:

1. .banner

  • Container class for the banner's outer div element.

2. .banner-column

  • Divides banner content into sections.
  • Vertically aligns content to middle.

3. .banner-image

  • Added to banner-column wrapper.
  • Contains icon or promotional image for banner.

4. .banner-body

  • Added to banner-column wrapper.
  • Contains main body copy for banner.

4. .banner-actions

  • Added to banner-column wrapper.
  • Contains any actions that the banner may require. Optional component.

The Banner includes the media-table object as defined by Citadel. This is similar to the OOCSS media object, but displays like a table to allow vertical centering of its content.
