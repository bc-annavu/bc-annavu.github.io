Illustrative Icons

Design icons are a set of icons that are more illustrative and informative than instructional, unlike the UI icons set. These are not designed to be used as actions, but more like cues to the user.

Unlike the UI icon set, not all of these icons are not vector based and as such, we unfortunately have 2 ways to call the icons in this set.

SVG based icons

Still the preferred way to include iconography into the Control Panel is via individual SVGs using the <icon /> component. Below are the collections of vector based design icons, with any special treatments they require. 

<icon glyph="ic-payment-mastercard" class="icon--medium"></icon>
<icon glyph="ic-payment-cvc" class="icon--medium"></icon>
<icon glyph="ic-payment-visa" class="icon--medium"></icon>
Notes:

Identical to UI Icons, the glyph attribute specifies which icon you would like, by name, from the list below. You should probably use icon--medium modifier for a slightly bigger size.

Sprite based icons

The Sprite directive is currently used only by the Flag icon set. These will be deprecated in an upcoming version, and this documentation will be changing soon. Please take this as reference for any current implementation, and not as instructions for development.

We created a rasterized image sprite to be used as a background image of our <sprite /> component. 

<sprite glyph="ic-australia"></sprite>
<sprite glyph="ic-united-kingdom"></sprite>
<sprite glyph="ic-new-zealand"></sprite>
Notes:

Similar to <icon />, the glyph attribute specifies which icon you would like, by name, from the list below.

flag icons

The Flag Icon set will be deprecated in an upcoming version. Please use this only as reference for current implementations. If your project requires Flag icons, please discuss this with the Design team and/or the UI Platform team.
