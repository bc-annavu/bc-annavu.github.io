Pill Group

A horizontal link list with a pill button as an active state. It is a visual treatment for tabs that are used to filter content within the table or panel. In mobile, the pill group is replaced by a select input.

  • One

HTML

CSS Classes:

1. .pillGroup

  • The parent element that wraps the pills.
  • Element is hidden in mobile screens.

2. .pillGroup-select

  • The input element that displays in mobile screens.

3. .pill

  • Can be either a <a> or <button>. By default, it displays as a gray text link.

4. .pill--active

  • The active modifier appended to .pill when segment is selected.
