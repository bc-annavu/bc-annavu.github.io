Style

Color

Brand and application colors.

Interface Icons

The BigCommerce control panel icon set is based on the open source Google Material Design icon set. It is imported via Bower, and the task gulp icons processes these into our folder structure for distribution and better naming.

Illustrative Icons

Design icons are a set of icons that are more illustrative and informative than instructional, unlike the UI icons set. These are not designed to be used as actions, but more like cues to the user.
