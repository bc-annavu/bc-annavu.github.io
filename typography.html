Typography

BigCommerce's primary typeface on the web is Source Sans Pro.

A cached version of the font stylesheet is included for most BC micro-applications in the template container. If the font is not readily available, either a new template container needs to be created to utilize the cached font stylesheet or the fonts can be hosted in the application. Do not use Google Fonts. Google Fonts will not work for internationalization.

Accessibility

Headings should be used for content structure. Screen readers and other assistive technologies allow the users to navigate web pages by heading structure so it is important to match heading tags with the content structure.

Pages should be structured in a hierarchical manner - where h1 is the most important (page titles, main content heading, appearing only once on a page) and h2 is the second most important and so on. Headings should also not be skipped (going from h1 to h4 for example).

Heading tags should not be used for text formatting or just visual appearance. If a design requires a particular visual treatment, use the text developer utility to adjust the visual formatting.
